# medicineapp_wx

此为 **本草善养** 小程序，账号中有后端分享代码 [后端分享链接](https://github.com/CSerxzm/medicineapp)，用于得到后端数据。

#### 功能

- 有获得中草药的相关信息
- 按季节、人群、节气获得养生知识
- 根据自己的状态，提供食疗方案
- 提供论坛功能，提供动态的发表和评论
